# Database Backup Automation

Herramienta simple en Python para automatizar backups de bases de datos.

## Funciones

- Exportación automática de bases de datos
- Organización de backups por fecha
- Compresión automática
- Registro de logs
- Automatización con cron

## Tecnologías

Python  
Bash  
Linux  
SQLite / MongoDB

## Uso

1. Configurar config.py
2. Ejecutar:

python backup.py

3. Programar con cron
